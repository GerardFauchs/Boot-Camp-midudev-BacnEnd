ESLINT:
Una vez instalado el paquete en modo dev, hace falta configurarlo, para entrar en
la guia de configuracion ejecutar el siguiente comando en Windows:
npm init @eslint/config
esto nos va preguntando para ir configurando ESLINT, finalmente crea
el archivo .eslinttrc.js con la configuración que hemos seleccionado.
Instalar la extension de VSC de: ES LINT.
Instalar la extension de VSC: Error Lens, para ver el error en la misma
linea al lado del codigo.

Configurar ESLINT para que corrija automaticamente los errores en guardar
el archivo:
ir a la carpeta oculta .vscode -> settings.json i agregar:
"editor.codeActionsOnSave": {
        "source.fixAll.eslint": true
    }
